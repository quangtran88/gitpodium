# Security

Report vulnerabilities privately via GitHub's
[Report a vulnerability](https://github.com/quangtran88/gitpodium/security/advisories/new)
form — please don't open a public issue for security bugs.

Only the latest `main` is supported. The generated `report.html` renders untrusted
data (git author names, GitHub API content), so injection/XSS reports about it are
especially welcome.
